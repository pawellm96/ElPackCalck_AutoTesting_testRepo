# ElPackCalc

Расчет электрических цепей для Autodesk Revit.

## Описание

Плагин для Revit, который выполняет расчет падения напряжения в электрических цепях.

## Требования

- Autodesk Revit 2025
- .NET Framework 4.8
- Visual Studio 2019 или новее

## Структура проекта

- `ElPackCalc` - основной проект плагина
- `ElPackCalc.Tests` - проект с автотестами (без запуска Revit)

## Тестирование

Проект включает автотесты, которые могут выполняться без запуска Revit, используя моки и рефлексию для тестирования бизнес-логики.

### Запуск тестов локально

```bash
dotnet test
```

### Запуск тестов в CI/CD

Тесты автоматически запускаются при push в репозиторий через GitHub Actions.

## Установка

1. Скопируйте скомпилированный `.dll` файл в папку Add-Ins Revit
2. Перезапустите Revit

### CI/CD

Проект настроен для автоматической сборки и тестирования через GitHub Actions. При каждом push в ветки `main`, `master` или `develop` автоматически:

1. Восстанавливаются зависимости
2. Собирается решение
3. Запускаются автотесты
4. Результаты тестов сохраняются как артефакты
